# tcc-marmita
Um app maneiro de marmitas


# Install
Last version Node.JS
Available: https://nodejs.org/en/download/


Create a app directory then:
//Install Facebook's react template app

npm install -g create-react-app

//Create a path to App
mkdir Project

//Install create-react-app components on App path
create-react-app Projeto

cd .\Project

//Install dependency Jquery
npm install jquery --save

//Install dependency PubSub-js
npm install pubsub-js --save

Copy files to src path
